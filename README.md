# Renda
Renda - это мощный кликер припасов для игры Танки Онлайн


## Установка
1. TamperMonkey - [click](https://www.tampermonkey.net)
2. Renda - [click](https://github.com/N3onTechF0X/some_TO_scripts/raw/main/Renda.user.js)
